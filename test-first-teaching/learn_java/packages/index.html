Packages

This lab is very straightforward. We need to create two classes, both named Comment. (One speaks English, and one German.) How do we keep the names straight? Packages!

Remember when dealing with packages that you should run all your tools (javac etc.) from the root directory -- not the sub-directory that actually contains the source files. See Learn Java Packages In 21 Seconds for "The Rules."

(Also, please note that using packages is not the best way to handle Localization or Internationalization of your application.)
